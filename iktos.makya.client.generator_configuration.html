class FineTuningGeneratorConfiguration(generator_type=None, additional_parameters=None, datasets=None, predictors=None, substructure_constraints=None, external_scorers=None)[source]¶

Bases: iktos.makya.client.generator_configuration.generator_configuration.GeneratorConfiguration

Configuration class for a Fine Tuning Generator. Needed to create a generator.

Parameters:

  • generator_type (Optional[str]) – Set by default. Not user specified

  • additional_parameters (Optional[dict]) – Additional parameters for generator configuration.

  • datasets (Optional[List[MakyaDataset]]) – Datasets to be used as chemical space. Can be a single molecule.

  • predictors (Optional[List[MakyaPredictor]]) – Predictors (predictor sets’s objectives) to guide generation.

  • substructure_constraints (Optional[List[SubstructureConstraint]]) – List of substructure constraints on the generated molecules. Set with the class SubstructConstraint.

  • external_scorers (Optional[List[MakyaScorer]]) – List of API configurations to score the generated molecules.

Return type:

None

Examples

>>> from iktos.makya.client.generator_configuration import FineTuningGeneratorConfiguration, MatchSubstructureConstraint
>>> datasets = await project.get_datasets()
>>> predictor_sets = await project.get_predictor_sets()
>>> predictors = await predictor_sets['my_predictor_set'].get_predictors()
>>> goals = [
...     MatchSubstructureConstraint(smarts="CC"),
... ]
>>> configuration = FineTuningGeneratorConfiguration(
...     datasets=[datasets['my-dataset']],
...     substructure_constraints=goals,
...     predictors=[predictors['Target1'], predictors['Target2']],
... )
>>> generator = await project.create_generator(generator_name="my-fine-tuning", configuration=configuration)

class FragmentGrowingGeneratorConfiguration(generator_type=None, additional_parameters=None, datasets=None, predictors=None, substructure_constraints=None, external_scorers=None, reference_molecule_smiles=None, exit_vector_params=None, final_product_filters=None, boost_search=False)[source]¶

Bases: iktos.makya.client.generator_configuration.generator_configuration.GeneratorConfiguration

Configuration class for a Fragment Growing generator. Needed to create a generator.

Parameters:

  • reference_molecule_smiles (Optional[str]) – SMILES of the fragment from which molecules are grown.

  • exit_vector_params (Optional[List[ExitVectorsParams]]) – Fragment exit vectors. Can be 1 or 2. To compute possible exit vectors, use the get_possible_exit_vectors() method. To draw possible exit vectors, use the draw_molecule_with_exit_vectors() function.

  • final_product_filters (Optional[List[CompoundFilter]]) – Filters on final generated molecules.

  • boost_search (Optional[bool]) – Activate the boost search option if set to True.

  • generator_type (Optional[str]) – Set by default. Not user specified

  • additional_parameters (Optional[dict]) – Additional parameters for generator configuration.

  • datasets (Optional[List[MakyaDataset]]) – Datasets to be used as chemical space. Can be a single molecule.

  • predictors (Optional[List[MakyaPredictor]]) – Predictors (predictor sets’s objectives) to guide generation.

  • substructure_constraints (Optional[List[SubstructureConstraint]]) – List of substructure constraints on the generated molecules. Set with the class SubstructConstraint.

  • external_scorers (Optional[List[MakyaScorer]]) – List of API configurations to score the generated molecules.

Return type:

None

Examples

>>> reference_smiles = 'Nc1ncnc2[nH]cnc12'

>>> from iktos.makya.client.ipython_console import draw_molecule_with_exit_vectors
>>> from rdkit import Chem
>>> vectors = await client.get_possible_exit_vectors(reference_smiles)
>>> draw_molecule_with_exit_vectors(Chem.MolFromSmiles(reference_smiles), vectors)

>>> from iktos.makya.client.generator_configuration import (
...     FragmentGrowingGeneratorConfiguration,
...     CompoundFilter,
...     ExitVectorsParams,
...     SubstructConstraint
... )
>>> commercial_compounds_filters = [
...     CompoundFilter(filter_name='MolWt', min=450, max=600),
...     CompoundFilter(filter_name='TPSA', min=0.33),
... ]
>>> product_filters = [
...     CompoundFilter(filter_name='QED', min=0.7),
...     CompoundFilter(filter_name='NumAromaticRings', max=3),
... ]
>>> commercial_compound_substruct_constraints = [
...     SubstructConstraint(
...         type='SUBSTRUCT_MATCH',
...         smarts='*~[#6]~[#6](=O)-[#8][H]'
...     )
... ]
>>> exit_vector_1 = ExitVectorsParams(
...     smiles=reference_smiles,
...     exit_vector=[0],
...     commercial_compound_filters=commercial_compounds_filters,
...     commercial_compound_substruct_constraints=commercial_compound_substruct_constraints
... )
>>> configuration = FragmentGrowingGeneratorConfiguration(
...     reference_molecule_smiles=reference_smiles,
...     exit_vector_params=[exit_vector_1],
...     final_product_filters=product_filters
... )
>>> generator = await project.create_generator(generator_name="my-fragment-growing", configuration=configuration)
